# Inventory Management System

## 📌 Description

The Inventory Management System is a web-based application developed using PHP and MySQL to help businesses manage their inventory efficiently. The system allows users to add, update, delete, and search products while maintaining accurate stock records. It provides a simple and user-friendly interface for managing inventory operations.

---

## 🎯 Objective

The main objective of this project is to simplify inventory management by providing a digital solution for tracking products, monitoring stock levels, and organizing inventory data efficiently.

---

## 🚀 Features

- User Login Authentication
- Dashboard
- Add New Products
- Update Product Details
- Delete Products
- Search Products
- View Product List
- Stock Management
- Database Integration
- Responsive User Interface

## ⚙️ Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into the `htdocs` directory.
4. Open phpMyAdmin.
5. Create a database.
6. Import the `inventory.sql` file.
7. Open your browser and visit:

```
http://localhost/Inventory-Management-System
```
